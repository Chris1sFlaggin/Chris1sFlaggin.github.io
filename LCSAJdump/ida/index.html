Drop ida_lcsajdump.py into ~/.idapro/plugins/. Press Ctrl+Shift+L to open the dockable gadget panel with live register filtering.

Available IDA 7.x · 8.x · 9.x x86_64 · ARM64 · RISC-V64

Features

Panel Capabilities

// FILTER

Live Search

Filter instantly by instruction text, sequential/jump-based types, and required registers using the sidebar UI. No re-analysis required.

// INTEGRATE

Native Docking

Built with PyQt5, the window docks directly into the IDA workspace. Drag it next to your disassembly view for side-by-side hunting.

// NAVIGATE

Sync View

Double-click any gadget row to immediately synchronize the main IDA disassembly view to that exact virtual address.

Workflow

Execution Flow

01

Load Target

Open the target ELF/binary in IDA Pro. Let the initial auto-analysis finish.

02

Configure Analysis

Press Ctrl+Shift+L. Set graph depth, min score, or enable --all-exec to scan all executable segments.

03

Filter & Hunt

Use the sidebar checkboxes to isolate gadgets modifying specific registers (e.g., rdi, rsi).
