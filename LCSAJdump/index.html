The Problem

The Blind Spot
of Linear Scanners

Traditional ROP scanners use a linear (sliding-window) approach over the executable's bytes. While effective for standard sequences, they fundamentally fail to discover gadgets that cross non-contiguous memory blocks via jumps.

LCSAJdump reconstructs the Control-Flow Graph (CFG) and identifies both types:

  • Contiguous Gadgets (Linear)

    Standard sequences ending with a control-flow transfer. Any scanner finds these.

  • Non-Contiguous Shadow Gadgets

    Complex chains spanning multiple Basic Blocks (LCSAJs), bypassing bad bytes via unconditional jumps or branches. Invisible to linear scanners.

// Trampoline Gadget — found by LCSAJ, invisible to ropper

0x4146a: ld a0, 8(s10)

0x4146e: jal -0x28974 ← Trampoline

0x18af2: c.mv a0, t3
0x18af4: bltz t3, 0x22e
0x18af8: c.ldsp ra, 0x58(sp)
0x18afa: c.addi16sp sp, 0x60
0x18afc: c.jr ra ← sink

The Solution

How LCSAJdump Goes Deeper

01

Graph
Reconstruction

Models the binary as a directed graph of Basic Blocks. Every control-flow edge is explicit.

02

Rainbow
BFS

Backward traversal with O(1) Early-Drop filter and Hard-Cap limits. Fast and exhaustive.

03

Shadow
Gadgets

Trampoline and conditional chains that bypass bad bytes. Invisible to linear scanners.

04

Dynamic
Scoring

Architecture-aware heuristics rank gadgets by register criticality and chain potential.

Core Evolution

ML Scoring Engine

> Il framework evolve da pesi empirici a un motore di ranking guidato dai dati.
> Sfruttando LightGBM LambdaRank e 23 feature strutturali, LCSAJdump trasforma la discovery dei gadget in un problema di Learning-to-Rank.

Level 1: Static Heuristic

Formula additiva manuale (magic numbers). Prestazioni deboli su binari complessi.

NDCG@1: 11.0%

Level 2: Optuna Calibrated

Coefficienti della formula core ottimizzati via 200 trial bayesiani (Optuna).

NDCG@5: 98.9% -> Altissima precisione a costo runtime zero.

Level 3: Deep ML Inference

Modello Ibrido (LightGBM) con feature semantiche (angr). Filtra il rumore sintattico.

NDCG@20: 100.0% -> Classifica infallibile per catene ROP complesse.

Get Started

Ready to Use

Install LCSAJdump via pip and analyze your binaries in seconds.

Install via pip:

pip install lcsajdump

Basic scan:

lcsajdump <path_to_binary>

Full example with all options:

lcsajdump -a auto -d 25 -k 10 -l 20 -s 5 -i 15 -v -o output.txt <path>

All options:

Flag Type Description
-a, --arch TEXT Target architecture (auto, riscv64, x86_64, arm64). Default: auto
-d, --depth INTEGER Max search depth in LCSAJ blocks. Controls chain length.
-k, --darkness INTEGER Pruning threshold — max visits per node. Higher = more gadgets, slower scan.
-l, --limit INTEGER Max number of gadgets to display in the output.
-s, --min-score INTEGER Minimum quality score for a gadget to appear in results.
-i, --instructions INTEGER Max number of instructions contained in a single node.
-v, --verbose FLAG Enable verbose output for detailed per-gadget results.
-o, --output PATH Write gadgets to the specified file path.
-b, --bad-chars TEXT Hex bytes to filter from gadget addresses (e.g. "000a0d").
--json FLAG Output gadgets as structured JSON instead of plain text.
--all-exec FLAG Analyze all executable sections, not just .text.
-al, --algo FLAG Use strictly the algorithmic ranking (bypass ML).
--viz FLAG Export full graph as JSON for 3D visualization.
--version FLAG Show the installed version and exit.
