The Problem

The Blind Spot of Linear Scanners

Traditional ROP scanners use a linear (sliding-window) approach over the executable's bytes. While effective for standard sequences, they fundamentally fail to discover gadgets that cross non-contiguous memory blocks via jumps.

LCSAJdump reconstructs the Control-Flow Graph (CFG) and identifies both types:

  • Contiguous Gadgets (Linear)

    Standard sequences ending with a control-flow transfer. Any scanner finds these.

  • Non-Contiguous Shadow Gadgets

    Complex chains spanning multiple Basic Blocks (LCSAJs), bypassing bad bytes via unconditional jumps or branches. Invisible to linear scanners.

// Trampoline Gadget — found by LCSAJ, invisible to ropper

The Solution

How LCSAJdump Goes Deeper

Graph Reconstruction

Models the binary as a directed graph of Basic Blocks using LCSAJ (Linear Code Sequence and Jump) analysis.

Rainbow BFS

Proprietary backward search algorithm starting from control-flow sinks to reconstruct full chains across block boundaries.

Shadow Gadgets

Identifies Trampoline and Conditional chains that bypass bad bytes — impossible to detect with a linear sliding-window.

Multi-Arch Support

Modular design natively supporting RISC-V 64GC, extendable to x86 and ARM via community forks.

Get Started

Ready to Use

Install LCSAJdump via pip and analyze your binaries in seconds.

Install via pip:

pip install lcsajdump

Basic scan:

lcsajdump <path_to_binary>

Full example with all options:

lcsajdump -a x86_64 -d 25 -k 10 -l 20 -s 5 -v <path>

All options:

Flag Type Description
-a, --arch TEXT Target architecture. Default: riscv64
-d, --depth INTEGER Max search depth in LCSAJ blocks. Controls chain length.
-k, --darkness INTEGER Pruning threshold — max visits per node. Higher = more gadgets, slower scan.
-l, --limit INTEGER Max number of gadgets to display in the output.
-s, --min-score INTEGER Minimum quality score for a gadget to appear in results.
-v, --verbose FLAG Enable verbose output for detailed per-gadget results.
--version FLAG Show the installed version and exit.
